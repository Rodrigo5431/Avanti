# 🚀 Portfólio de Projetos: Landing Page & GitHub API

Este repositório contém dois projetos distintos desenvolvidos para demonstrar habilidades em estruturação de interfaces responsivas e consumo de APIs externas.

---

## 🛒 Projeto 1: Avanti - E-commerce Landing Page

Uma landing page completa e responsiva para um e-commerce fictício ("Avanti"). O layout foi construído com foco na experiência do usuário (UX) e adaptabilidade para múltiplas telas.

### ✨ Funcionalidades
* **Mega Menu Interativo:** Navegação por departamentos e categorias com dropdown fluído.
* **Carrosséis Dinâmicos:** Vitrines de "Lançamentos" e "Mais Vendidos" implementadas com a biblioteca **Swiper.js**.
* **Design Totalmente Responsivo:** Layout fluido adaptável para Desktop, Tablet e Mobile, utilizando CSS Flexbox e Media Queries.
* **Interatividade Mobile:** Menu hambúrguer funcional e rodapé em formato de acordeão expansível desenvolvidos com JavaScript puro.
* **Tipografia Customizada:** Integração com Google Fonts (Montserrat, Nunito e Nunito Sans).

### 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3 (Variáveis CSS, Flexbox avançado, Media Queries)
* JavaScript (ES6+, DOM Manipulation)
* Swiper.js (Carrossel)

## 🐙 Projeto 2: GitHub API Integration

Uma aplicação web desenvolvida para consumir a API REST pública do GitHub, permitindo a busca e exibição em tempo real de dados de desenvolvedores.

### ✨ Funcionalidades
* **Busca de Usuários:** Campo de pesquisa para localizar perfis no GitHub através do *username*.
* **Exibição de Informações:** Renderização dinâmica de dados do perfil (nome, foto, bio).
* **Tratamento de Dados:** Requisições assíncronas tratadas de forma segura com feedback para o usuário caso o perfil não exista.

### 🛠️ Tecnologias Utilizadas
* HTML5 & CSS3
* JavaScript (Fetch API, Promises, Async/Await)
* Node.js / NPM

## ⚙️ Como executar os projetos

1. Clone o repositório em sua máquina local:
   ```bash
   git clone [https://github.com/Rodrigo5431/Avanti.git](https://github.com/Rodrigo5431/Avanti.git)
   ```

### Avanti - E-commerce Landing Page
Acesse o diretório do projeto e abra o arquivo principal:
```bash
cd Avanti-LandingPage
```
Após acessar a pasta, abra o arquivo `index.html` diretamente no seu navegador.

### GitHub API Integration
Acesse o diretório do projeto, instale as dependências e inicie o servidor:
```bash
cd Avanti-Github
npm install
npm run dev
```
